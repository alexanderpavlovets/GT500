Меню сайта
Бывает позиционирование в нормальном потоке (так называется размещение элементов 1за1 по вертикали - стандартно в HTML).
Но мне нужно абсолютное - позиционирование относительно "родителя", в этом случае от body (позиционировать буду контент).
Позиционирования (position) бывают:
  • static - нормальный поток
  • absolute - абсолютное - относительно родителя
  • relative - относительное - относительно нормального потока
  • fixed - фиксированное позиционирование
При абсолютном - надо показать где разместить блок, относительно родителя:
  • left - указывает на сколько надо сместить блок относительно левого края
  • right - указывает на сколько надо сместить блок относительно правого края окна
  • top - указывает на сколько надо сместить блок относительно верхнего края окна
  • bottom - указывает на сколько надо сместить блок относительно нижнего края окна
Этот попробую вместить вправо внутри контента - сам. Вместил, а текст ,собака, из родителя залазит. Паддингом попробую убрать в родителе.
Паддинг влияет на размер блока - не вышло, т.е. задаю паддинг - увеличивается общий размер(ширина уазанная влияет только на содержимое). Я бы вставил ещё один div для контекста, но урок этого не предпологает. На этом всё.
