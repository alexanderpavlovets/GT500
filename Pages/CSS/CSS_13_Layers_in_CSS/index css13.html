Слои в CSS

Слои пронумеровать с помощью свойства z-index (1-6), чем выше номер - тем выше слой.
Слои часто используются для создания раскрывающихся меню. Наводишь мышью на содержимое меню - появляется слой, соответствующий контенту на котором курсор, остальные слои остаются прозрачными.

Буду делать пример такого списка (тут смотри в КСС - там многое):
Книги Фильмы Музыка Игры
Создал 4 div - основные элементы меню, потом создал подразделы - тоже divы, расположил их всех абсолтным позиционированием - что бы можно было указать точно где их отображать потом.








Кучу br наставил, поскольку позиционирование абсолютное - нет времени делать всё красиво.

После указания места для div-ов подменю указываю свойство в CSS: z-index - это определит номер слоя.

Что бы слои открывали при наведении курсора на соответствующий раздел - надо сделать из прозрачными - добавить visibility:hidden. После этого элементы становятся невидимыми.
Дальше на CSS нельзя ничего сделать, потому что CSS отвечает за размещение элементов - необходим JS (не обязательно он, но язык нужен).
Напишу маленький скрипт на JS и подключу его к этой странице, потом укажу в ссылках меню, что надо включать функцию при наведении мыши.
Не скрываются слои (подменю) при отводе мыши - потому что нет функции такой в JS.
