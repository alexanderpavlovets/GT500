Урок 4 - псевдоэлементы и псевдоклассы

Псевдоэлементы:

Есть несколько элемнтов, которые не существуют в HTML, но они есть на странице:

  • Первая буква слова
  • Первая строка абзаца

Это и есть псевдоэлементы, им можно задавать стиль в CSS:

  • p:first-letter {....;}
  • p:first-line {.....;}

Псевдоклассы:

Псевдоклассы помогают работать с ссылками в документе. Ссылки имеют 4 состояния:

  • простая
  • активная
  • посещённая
  • на которую наведён курсор
Для каждого из этих состояний есть свой псевдокласс:
  • a:link - задает стиль обычной ссылки
  • a:active - задает стиль активной ссылки
  • a:visited - задает стиль посещенной ссылки
  • a:hover - задает стиль ссылки, на которую наведен курсор
Эти псевдоклассы и помогают работать с ссылками - смотри в CSS.

Урок 5 - свойства color и background

Цвет - color

Свойство color - наследуемое, если в body задать color - все элементы будут этого цвета. Для того, что бы поменять цвет элемента - надо задать ему свой собственный цвет.
Задать цвет можно 3-мя способами:
  • именной - red
  • шеснадцатиричный #FF0000
  • десятичный в модели RGB 255,0,0

Фон - background

При помощи CSS можно задавать фон не только странице, но и любому элементу на странице. Как обычно в CSS задаёшь правило для элемента - в него фон. Можно и классы и ай-дишники использовать.
Brackets с Emmet всё пишут буквально сами - очень продумано, тем не менее, надо рассмотреть группу свойств background:
  • background-color: - цвет
  • background-image:url(/path) - картинка для фона
  • background-repeat:repeat/no-repeat/repeat-x/repeat-y - повторение фонового изображения: да/нет/по горизонтали/ по вертикали
  • background-attachment: fixed/scroll - фоновое изображение фиксировано при прокрутке т.е. скролится только текст/прокручивается вместе с текстом (по умолч.). Для всех привычней - scroll
  • background-position:center(left/right) top(center/bottom) или в % или в пикселах - задаёт расположение фонового изображения. При задании формата в % или пикселах указываешь насколько отступает верхний левый угол, сначала горизонтально, потом вертикально.

Сокращенная запись свойств

В CSS для многих свойств существует сокращенная запись. Для этого можно через пробел перечислить значения свойств: 
    body{
    background:url(/path) no-repeat blue center top;
    }
