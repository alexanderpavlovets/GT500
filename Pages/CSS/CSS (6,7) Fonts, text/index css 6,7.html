Урок 6 - Шрифты в CSS

Для задания параметров шрифтов в CSS используется свойство font

Font-family

Это свойство CSS задаёт гарнитуру шрифта. Все шрифты услоно делятся на несколько групп (гарнитур):
  • Serif - шрифты с засечками, например, Times New Roman
  • Sans-serif - шрифты рубленные, без засечек, например, Arial
  • Monospace - моноширинные шрифты, например, Courier New
  • курсивные шрифты, например, Calisto MT
  • Fantasy - декоративные шрифты, например, Torhok
Для примера шрифтов:

В качестве значения можно указать шрифт и группу шрифтов:
font-family: Verdana, sans-serif;
Это означает, что при отображении страницы будет выбран указанный шрифт, а если такого шрифта не найдётся у пользователя - будет выбран шрифт из указанного семейства.
Можно при указании значений свойства font-family указать несколько шрифтов - в порядке приоритета, через запятую.

Стиль шрифта:
  • normal - обычный
  • oblique - наклонный
  • italic - курсивный

  • normal - обычный

Font-weight

Задаёт ширину букв шрифта, значения:
  • normal
  • bold
  • bolder - более жирный по отношению к базовому, унаследованному от предка
  • lighter - менее жирный по отношению к базовому, унаследованному от предка
Так же используются числовые значения: 100,200,300,400 (normal),500,600,700 (bold),800,900

Font-size

Задавать размеры в CSS можно тремя способами:
  • с помощью ключевых слов (xx-small, x-small, small, medium, large, x-large, xx-large, smaller, large)
  • с помощью относительных единиц (% и em)
  • с помощью единиц измерения длины (пикселы, пункты, сантиметры и миллиметры)
Ключевые слова использовать не рекомендуется, т.к. разные браузеры отображают их по-разному.
Относительные единицы - задают размеры относительно элемента-предка
Единицы измерения - наиболее удобный на данный момент вариант. Существует:
  1. Три относительные единицы измерения
    • px - равен одной точке на экране, называемой пикселом
    • em - равен высоте шрифта, используемого в данном элементе
    • ex - равен высоте строчной буквы "х" шрифта, используемого в данном элементе
  2. Пять абсолютных едениц измерения:
    • in - равен 1 дюйму (2,54 см)
    • pt - равен 1/72 дюйма
    • pc - равен 1/6 дюйма
    • mm - равен 1 миллиметру
    • sm - равен 1 сантиметру
Поскольку у пользователей разные разрешения экранов, рекомедуется использовать 3 единицы измерения:
  • pt - для фиксированного дизайна
  • % - для резинового дизайна
  • em - для пропорционального изменения размера

Сокращённая запись


В этом случае значения всех свойств перечисляются через пробел в следующем порядке: font-style, font-variant, font-weight, font-size, font-family. Причем, любое из свойств, кроме font-size и font-family, можно не указывать. Давайте зададим нашим параграфам следующие свойства: 
 
    p#sn {font: italic 12px Verdana, sans-serif}

Урок 7 - Текст

Text-decoration

Отвечает за оформление текста, принимает значения:
  • none - у текста нет оформления
  • underline - текст подчёркивается
  • overline - текст надчёркивается над линией, расположенной над текстом
  • line-through - зачёркнуты текст
  • blink - мигающий текст(мало где работает)

Text-intend

Отвечает за отступы в абзацах! Отступ задаётся в % и единицах измерения.
Отступ может быть как положительный (отступ) так и отрицательный (висячая строка)

Text-transform

Отвечает за смену регистра текста. Принимает значения:
  • capitalize - меняет первую букву каждого слова на заглавную
  • uppercase - меняет все буквы на заглавные
  • lowercase - меняет все буквы на строчные
  • none - изменений не происходит

id=tt_cap И каждая первая буква слова должна изменится на заглавную

id=tt_up - всё должно быть в верхнем регистре

id=tt_lc - всё должно быть в нижнем регистре

Интервалы

В CSS есть несколько свойств регулирующих расстояния между словами/буквами в словах/строками:
  • word-spacing - задает интервал между словами
  • letter-spacing - задает интервал между буквами
  • line-height - задает интервал между строками
Значения задаютсяв % и в единицах измерения.
