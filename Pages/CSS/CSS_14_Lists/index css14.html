Урок 14 - Списки

CSS предоставляет больше возможностей при создании списков, чем HTML. Для списков существует всего 3 свойства:
  • list-style-type - определяет внешний вид маркера или нумератора
  • list-style-image - определяет пользовательское изображение маркера
  • list-style-position - определяет положение маркеров относительно блока
Примеры:

list-style-type

Для маркированных списков, всё аналогично HTML:
  • disk - закрашенный кружок
  • circle - незакрашенный кружок
  • square - закрашенный квадрат
Примеры маркированного списка:
  • Первый пункт
  • Второй пункт
  • Третий пункт
  • Первый пункт
  • Второй пункт
  • Третий пункт
  • Первый пункт
  • Второй пункт
  • Третий пункт
Для нумерованного списка перечень более широк, но не все значения поддерживаются всеми браузерами.
Рекомендуется использовать:
  • decimal - десятичные числа
  • lower-roman - строчные римские цифры
  • upper-roman - прописные римские цифры
  • lower-alpha - строчные латинские буквы

list-style-image

Вместо маркера устанавливается картинка, для этого в CSS #spisok {list-style-image:url(/path);}
  • Первый
  • Второй
  • Третий
Запомеи - на внешний вид списка влияет размер картинки.

list-style-position

Это свойство определяет положение маркера:
  • inside - внутри блока
  • outside - снаружи блока
Делаю 2 списка в div-е, что б увидеть разницу:
  • Первый первого
  • Второй первого
  • Третий первого
  • Первый второго
  • Второй второго
  • Третий второго
Разница: маркер стоит в границе основного блока списка и за ней.

Сокращенная запись свойства list-style

Как обычно - все значения через пробел, обязательного порядка не указано, но пример такой вот:
#spisok { list-style: circle url(/path) inside; }
