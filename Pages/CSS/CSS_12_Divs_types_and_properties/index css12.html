display:none

Это значение убирает элемент со страницы. Часто исползуется для раскрывающихся меню сайтов, с помощью языка програмировани (напромер JS). Это меню, при щелчке на которое раскрывается список подпунктов.
Существует охожее свойство в CSS - visibility:visible/hidden.
Свойства похожи, но есть различия:
  • visibility:.. - делает элемент прозрачным
  • display:none - скрывает элемент как-будто его и не было

  • visible - если содержимое превышает размеры блока, оно все-равно останется на экране. Результат будет такой же, как на рисунке выше.
