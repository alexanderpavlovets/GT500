Урок 15 - навигационные карты - map

Буду делать навигационную карту из картинки ниже. Пылесос - google, машинка - google/translate страница, холодильник - google/maps.

Задаю тег map с атрибутом name - для создания карты, имени её .... что б потом на карту ссылатся.

После вношу ссылку в картинку на образованную тегом map карту - в img атрибут usemap="#map_name"

Для описания конкретных областей внутри тегов map , испольуется тег area со следующими атрибутами:

  • shape - определяет форму области, может принимать следующие значения:
    • rect - область в виде прямоугольника
    • circle - область в виде круга
    • poly - область в виде многоугольника
    • default - вся область
  • cootds - задает координаты отдельной области:
    • для rect задаются координаты верхнего левого и правого нижнего углов прямоугольника
    • для circle задаются координаты центра круга и радиус
    • для poly задаются координаты вершин многоугольника в нужном порядке
  • href - определяет адрес ссылки
  • target - применяется при использовании фреймов и указывает фрейм в который нужно загрузить страницу
  • alt - задает альтернативный текст для области

Для задания координат сещуствую специальные программы - сейчас сам "на глаз" задам что б понимать принцп работы просто. А проги же умеют обводить любые области и сами описывают их. Тут для примера - ImageReady программа.
