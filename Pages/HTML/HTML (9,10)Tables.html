Параметры таблиц: отступ, ширина, цвет фона, рамка

У тега table есть много атрибутов:

  • width - задаёт ширину таблицы ( в пикселях/%
  • bgcolor - задаёт цвет фона ячеек таблицы
  • background - заливает фон таблицы рисунком
  • border - задаёт рамку указанной ширины вокруг всей табилцы
  • cellpadding - задаёт отступ в пикселах мжду границей клетки и её содержимым
  • align - выравнивает таблицу на странице

Есть и ещё параметры:

  • cellspacing - задет расстояние между ячейками в пикселах
  • cellpadding - задаёт расстояние между текстом и внутренней границей ячейки таблицы в пикселах

В этой таблице cellspacing установлен на 0 - поэтому границы в привычном варианте, если поставить больше чем 0 - будут ганицы двойные (как в пред. таблицах).

Для границ есть собственные параметры:

  1. frame - задаёт вид рамки вокруг таблицы и принимает значения:
    • void - нет рамки
    • above - только верхняя рамка
    • below - только нижняя рамка
    • hsides - верхняя и нижняя рамки
    • vsides - правая и левая рамки
    • lhs - левая рамка
    • rhs - правая рамка
    • box - все четыре части рамки
  2. rules - задаёт вид внутренних границ таблицы и может принимать значения:
    • none - нет границ между ячейками
    • groups - только между группами строк и группами столбцов
    • rows - между строками
    • cols - между столбцами
    • all - все границы между ячейками

Работа с ячейками - атрибуты для tr и td:

Таблицы формируются построчно - атрибуты заданные tr распростроняются на всю строку, бывают:

  • align - right/left/center
  • valign - top/bottom/middle
  • bgcolor - цвет указывает

Не тупи - тут все ясно

Таблица с форматированием строк

1112

Эти же параметры можно применять и к ячекам, т.е. любому тегу td можно задать тоже самое и ещё 2:

  • width - ширина в пикселах или %, меняется ширина всего столбца (за 100% берётся ширина таблицы)
  • heigth - высота, задав для ячейки - высота задаётся для всей строки
Таблица с форматированием ячеек

1112

Если не задавать ширину и высоту - таблица будет форммироватся по содержимому!
