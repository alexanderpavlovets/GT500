Задача усложнённая

Пользователь должен не выбирать день недели а вводить его в текстовое поле

Создаю форму с 3-мя элементами: input-type"text", input="button" , textarea. Присваиваю каждому елементу имена и как всегда привязываю функцию к кнопке при онКлик



Что бы не писать много-много if буду использовать функцию switch

        switch (B)
            {
            case L1: S1;
            case L2: S2;
            ... ... ...
            case Ln: Sn;
            default:S
            }

Работает это так: вычисляется значение выражения B. Если B равно L1, то выполняется оператор S1. Если B равно L2, то выполняется оператор S2 и т.д. Если B не равно ни одному из значений, то выполняется оператор S.

Так же добавляю break; к конце каждого условия что бы прирывать функцию - так лучше для производительности
