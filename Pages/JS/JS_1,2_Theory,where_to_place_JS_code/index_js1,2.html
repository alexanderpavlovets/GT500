Урок 1 - Основные понятия

Язык программирования JS от Netscape + Sun Microsystems анонсирован в 1995 году. JS предназначен для создания интерактивных html-документов.


Основные области применения JS:
  • Создание динамических страниц, т.е. страниц, содержимое которых может меняться после загрузки
  • Проверка правильности заполнения пользовательских форм
  • Решение "локальных" задач с помощью сценариев
  • JavaScript-код - основа большинства Ajax-приложений

JS позволяет создавать приложения, выполняемые на стороне клиента, т.е. эти приложения выполняются браузером на копьютере пользователя.

Программы (сценарии) на этом языке обрабатываются встроенным в браузер интерпретатором. Не все сценарии выполняются корректно во всех браузерах, поэтому надо тестировать во всех браузерах

JS регистрозависимый - т.е. заглавные и прописные буквы алфавита - это разные символы.

Самые базоыве понятия: литералы, переменные и выражения

Литералы:

Литералы - это простейшие данные, с которыми может работать программа.

  • Литералы целого типа - целые числа в представлении:
    • десятичном, например: 15, +5, -174
    • шестнадцатеричном, например: 0х25, 0хff. Шестнадцатеричные числа включают цифры 0 - 9 и буквы a, b, c, d, e, f. Записываются они с символами 0х перед числом
    • восьмеричном, например: 011, 0543. Восьмеричные числа включают только цифры 0 - 7
  • Вещественные литералы - дробные числа. Целая часть отделяется от дробной точкой: 99.15. Возможна экспоненциальная форма записи , например: 2.73е-7. В привычном виде это число 2.73Х10 -7, но в JS знак умножения и 10 заменяются символом "е"
  • Логические значения - из двух: true/false - истина/ложь
  • Строковые литералы - последовательность символов, заключённая в одинарные или двойные кавычки. Например "Ваше имя"

Переменные:

Переменные используются для хранения данных. Определяются оператором - var, после которого следует имя переменной. Имя переменной должно начинатся с буквы латинского алфавита или с символа подчёркивания. Само имя может включать буквы латинского алфавита, цифры и знак подчёркивания.

Например:
var test
var _test
var _my_test1

Каждой переменной можно присвоить значение либо при её инициализации (объявлении), либо в коде самой программы. Оператор присваивания обозначается знаком "=", но знак равенства не имеет здесь своего прямого значения. В данном случае, он лишь указывает на то, что данной переменной присвоили значение.

Например:
var a=15
var b=23.15
var c='выполнено'
var s=true

Каждая переменная имеет тип, определяемый значением переменной. Так в примере:
  • a и b имеют тип number
  • c - тип string
  • s - тип логический

Выражения:

Выражения состоят из литералов, переменных, знаков операций и скобок. В результате вычисления выражения полкчается единственное значение, которое может быть числом, строкой или логичеким значением.
В выражении a*b, a и b называются операндами, а знак * - знак операции. В JS определены следующие операции:

  • + - сложение
  • - - вычитание
  • * - умножение
  • / - деление
  • % - остаток от деления целых чисел
  • ++ - увеличение значения операнда на единицу
  • -- - уменьшение значения операнда на единицу

Операторы в выражении вычисляются слева направо, в соответствии с приоритетами арифметических операций. Изменить порядок вычисления можно с помощью скобок.

Для каждого из арифметических операторов есть форма, в которой одновременно с заданной операцией выполняется присваивание:

  • операция - эквивалентный оператор присваивания
  • X+=Y - X=X+Y
  • X-=Y - X=X-Y
  • X*=Y - X=X*Y
  • X/=Y - X=X/Y
  • X%=Y - X=X%Y

В данном случае, сначала вычисляется правый операнд, а затем полученное значение присваивается левому операнду.

Для того, что бы можно было сравнивать два значения в JS определены операции сравнения, результатом которых может быть только логическое значение: true/false:

  • < - меньше
  • > - больше
  • == - равно
  • != - не равно
  • <= - меньше или равно
  • >= - больше или равно

В JS определены логические операции:

  • && - логическое И (AND)
  • || - логическое ИЛИ (OR)
  • ! - логическое НЕ (NOT)

Для строковых литератов определена операция конкатенация строк, т.е. их объединение. Обозначается знаком "+", результат тоже строка.
Пример:
var st1="Привет";
var sr2="Вася";
var st3=st1+st2;
В результате в переменной st3 будет значение "ПриветВася"

Урок 2 - где размещать JS-код

JS-код может быть размещён как на самой странице, так и во внешнем файле.

JS-код на странице:

Сценарии написанные на языке JS могут располагатся на самой странице html между тегами <script></script>, которые в свою очередь обычно находятся в тегах <head></head>. У тега <script> должен быть указан параметр (атрибут) language, который и указывает язык написания скрипта:

<html> <head> <title>Заголовок документа</title> <link rel="stylesheet" type="text/css" href="style.css"> <script language="javascript"> </script> </head> <body> </body> </html> Теоретически теги <script></script> могут располагатся и в теле документа (в тегах <body></body>).Выполнятся такой сценарий будет тогда, кгда браузер "дочитает" страницу до этого места (до тегов <script></script>). Вроде бы такой способ не прринято использовать.

JS-код во внешнем файле:

Стандартное внешние подключение JS в тегах <head></head> с помощью тега <script></script> <html> <head> <title>Заголовок документа</title> <link rel="stylesheet" type="text/css" href="style.css"> <script type="text/javascript" src="script.js"></script> </head> <body> </body> </html>
