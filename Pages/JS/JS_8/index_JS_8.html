Lesson 8 Switch operator

Условие задачи:
Мы знаем прогноз погоды на неделю вперед, но пользователю будем выводить только один день - тот, который пользователь себе выберет.
Решение - первый раз буду использовать IF
Создаю форму сс списком дней недели, кнопкой "ок" и полем для вывода результатов это теги: 

 form , select, input type="button", textarea

После чего пишу функцию в JS файле, которая выведет прогноз погоды в зависимости от выбранного дня, после нажатия кнопки. Соответственно - функция вызывается onClick на кнопке

 function Vday(obj){

   obj.res.value="Во вторник будет солнечно, температура воздуха поднимется до +25 градусов";

    }

Надо помнить про метод выбора [0][0], эта запись означает, что выбран первый елемент первого елемента - смешною
Вобщем - форма имеет 3 елемента: список, кнопку, текстовое поле. Запись [0][0] означает что выбираем первый елемент формы(список) + первый елемент из этого елемента(понедельник). ".selected" отмечает что елемент выбран.
